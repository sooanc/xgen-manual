# 일반 사용자

> 회사가 미리 배포해둔 Agent를 **채팅으로 활용**하는 사용자. 사용 가능한 메뉴는 **Agent 채팅 / 기술지원 / 대시보드** 셋입니다. 그 외 메뉴(Agent 제작·도구 연동·지식관리)는 Agent 개발자 권한이 필요합니다.

## Agent와 채팅하고 싶다
- 시작 위치: 좌측 사이드바 **Agent 채팅 → 새 채팅**
- 절차: [채팅 사용](../user/14-chat.md)

## 로그인 후 메인 화면 보기 (대시보드)
- 시작 위치: 로그인 직후 자동 진입 (또는 좌상단 **XGEN** 로고 클릭)
- 절차: [대시보드](../user/18-dashboard.md)
- 표시 범위: 일반 사용자 권한을 가진 모든 사용자에게 동일하게 노출되는 메인 화면. 공지·FAQ·자주 묻는 질문 등 일반 사용자 영역의 기본 콘텐츠를 모아 보여줌. 거버넌스·시스템 운영 위젯은 해당 권한이 있는 사용자 대상으로 노출됨.

## 공지·FAQ를 찾거나 문의하고 싶다
- 시작 위치: 좌측 사이드바 **기술지원**
- 절차: [기술지원](../user/19-tech-support.md)

## 솔루션 화면이 처음이다, 어디부터 봐야 하나
- [시작하기](../user/10-getting-started.md) → [로그인](../user/11-login.md)
