# 거버넌스 담당자

> AI 사용에 대한 **위험·통제·감사**를 책임지는 사용자. 일반 시스템 관리자와 권한이 분리되는 게 일반적입니다.

## 로그인 직후 내 역할에 맞는 메인 화면을 보고 싶다
- 시작 위치: 로그인 직후 자동 진입 (또는 좌상단 **XGEN** 로고)
- 절차: [대시보드 · 거버넌스 담당자 뷰](../admin/30-dashboard.md#거버넌스-담당자-뷰)
- 거버넌스 담당자(`admin.governance:*` 권한)에게는 **위험 등급 정책 현황**·**금칙어 정책 현황**·**승인 대기 Agent 큐** 등 거버넌스 전용 위젯이 메인 화면에 노출됩니다.

## 시스템 관리자가 1차 승인한 에이전트를 거버넌스 관점에서 검토·승인하려면 (2차 거버넌스 승인)
- 시작 위치: 관리 설정 → **AI 거버넌스 → 에이전트플로우 승인**
- 절차: [AI 거버넌스 · 에이전트 승인](../admin/29-governance-dashboard.md#agent-approval)
- 흐름: 시스템 관리자 1단계([Agent 관리 — 배포 승인](../admin/32-agent-operations.md#agent-mgmt-deploy-approval)) 를 통과한 에이전트만 본 큐에 노출됩니다. 위험도 평가에서 임계치 초과로 강제 등록된 항목도 같은 큐에 합류.
- 큐 보기: 상단 4개 stat 카드(**전체 / 대기 / 승인 / 반려**) 클릭으로 즉시 필터.
- 검토 항목: 노드 구성·작성자·부서·파라미터(PII 노출 가능 노드 우선)·카테고리.
- 결정: **승인** (코멘트 선택) 또는 **반려** (코멘트 *필수*). 둘 다 `governance_reviewed_by` + `governance_review_comment` + 일시가 [AI 서비스 변경 이력](../admin/29-governance-dashboard.md#audit-tracking) 에 영구 기록됩니다.
- 결과: 승인 즉시 사용자 검색·실행에 노출. 반려 시 작성자가 사유를 확인하고 수정한 뒤 0단계부터 재요청.

## 거버넌스 승인 큐가 비정상적으로 쌓이고 있다 (병목 확인)
- 시작 위치: [대시보드 · Agent 배포/승인 상태](../user/18-dashboard.md) 위젯의 *거버넌스 승인 대기* 카운트
- 절차: 카운트 클릭 → [에이전트 승인](../admin/29-governance-dashboard.md#agent-approval) 큐 진입 후 *대기* stat 카드 필터로 일괄 처리
- 운영: 위젯에 일정 건수 이상 쌓이면 시스템 관리자(1단계)는 정상 작동하나 거버넌스 단계가 병목이라는 신호. 주간 단위 큐 점검을 권장합니다.

## 위험도 평가 카테고리·가중치를 조정하려면
- 시작 위치: 관리 설정 → AI 거버넌스 → AI 통제 정책 관리
- 절차: [AI 거버넌스 - 통제 정책 관리](../admin/29-governance-dashboard.md#control-policy)
- 권장: 분기 1회 재조정

## PII 마스킹 규칙을 추가하거나 조정하려면
- 시작 위치: 관리 설정 → AI 거버넌스 → AI 통제 정책 관리 → PII 정책
- 절차: [PII 보호 정책](../admin/25-pii-policy.md)

## 거버넌스 정책·승인 변경 이력을 추적하려면
- 시작 위치: 관리 설정 → AI 거버넌스 → AI 서비스 변경 이력
- 절차: [AI 거버넌스 - AI 서비스 변경 이력](../admin/29-governance-dashboard.md#audit-tracking)
- 거버넌스 한정 감사 (일반 시스템 로그와 분리)

## 정기 점검 일정을 등록·관리하려면
- 시작 위치: 관리 설정 → AI 거버넌스 → AI 점검 이력 및 계획
- 절차: [AI 거버넌스 - 점검 이력 및 계획](../admin/29-governance-dashboard.md#inspection)

## 거버넌스 종합 모니터링이 궁금하다
- 시작 위치: 관리 설정 → AI 거버넌스 (기본 진입 화면)
- 절차: [AI 거버넌스](../admin/29-governance-dashboard.md)
