# AI 거버넌스 대시보드

본 챕터는 AI 사용 현황·위험·컴플라이언스 준수 상태를 한눈에 모니터링하는 **AI Governance Insight Dashboard** 사용법을 다룹니다.

## 거버넌스 대시보드란

거버넌스 대시보드는 다음 질문에 답하는 운영 지표를 시각화합니다.

- 우리 조직은 AI를 어떻게·얼마나 사용하고 있나?
- 위험한 사용 패턴(PII 노출, 금칙어 위반 등)이 있는가?
- 규제·내부 정책 준수 현황은?
- 어떤 사용자·부서가 가장 활발히 사용하나?

## 화면 진입

좌측 메뉴 **AI 거버넌스** 또는 대시보드 상단 **AI Governance Insight Dashboard** 위젯을 선택합니다.

![AI 거버넌스 모니터링 — 위젯 그리드로 구성된 종합 대시보드](images/admin-gov-monitoring.png)

## 주요 위젯

### 사용 현황 (Usage)

| 위젯 | 표시 내용 |
|---|---|
| 일/주/월 활성 사용자 (DAU/WAU/MAU) | 시간 단위별 활성 사용자 수 추이 |
| 채팅 메시지 수 | 일/주/월 메시지 추이 |
| 에이전트플로우 실행 수 | 배포된 플로우의 호출 횟수 |
| 토큰 소비량 | LLM 호출의 입력·출력 토큰 누적 (비용 추적용) |

### 위험 지표 (Risk)

| 위젯 | 표시 내용 |
|---|---|
| 위험 등급 분포 | 최근 기간의 위험 등급별 이벤트 비율 (낮음/보통/높음/위험) |
| PII 탐지 빈도 | 정책별 탐지 횟수 추이 |
| 금칙어 위반 | 차단된 입력·응답 횟수 |
| 의심스러운 패턴 | 비업무 시간 접속, 대량 다운로드 등 |

### 컴플라이언스 (Compliance)

| 위젯 | 표시 내용 |
|---|---|
| 정책 적용 현황 | 활성/비활성 정책 수 |
| 감사 로그 보존 충족률 | 규정상 보존 기간 대비 실제 보존 데이터 비율 |
| 권한 검토 현황 | 마지막 권한 검토 시점 (90일 초과 시 경고) |
| 데이터 분류 적용률 | 컬렉션 중 분류 라벨이 부여된 비율 |

## 위험 평가 카테고리

위험 평가에 사용되는 카테고리와 가중치를 설정할 수 있습니다.

| 카테고리 | 영문 | 설명 |
|---|---|---|
| PII 노출 | PII Exposure | 개인정보 마스킹 실패·우회 |
| 데이터 외부 반출 | Data Exfiltration | 대량 다운로드·외부 메일 전송 |
| 권한 오용 | Privilege Misuse | 비정상 권한 상승 사용 |
| 비정상 접근 | Abnormal Access | 비업무 시간·해외 IP 접근 |
| 정책 위반 | Policy Violation | 금칙어 사용·차단된 도구 호출 |

각 카테고리는 **가중치(Weight)** 를 가지며, 종합 위험 점수 계산에 반영됩니다.

## 가중치 조정

조직마다 위험 우선순위가 다르므로 가중치를 조정합니다.

1. 거버넌스 대시보드 우상단 **설정 (⚙)** 버튼
2. 카테고리별 가중치 슬라이더 (0~10)
3. 합산이 자동으로 정규화됨
4. **저장**

!!! note "가중치 조정 화면 캡처는 별도 갱신 예정"
    설정(⚙) 버튼 클릭 시 노출되는 카테고리별 가중치 슬라이더 화면 캡처는 다음 회차에 보강 예정입니다.

!!! info "권장 가중치 (금융권 예시)"
    - PII 노출: 10 (최우선)
    - 데이터 외부 반출: 9
    - 권한 오용: 8
    - 정책 위반: 6
    - 비정상 접근: 5

## 기간 비교

대부분의 위젯은 **기간 비교** 모드를 지원합니다.

- 이번 달 vs 지난 달
- 이번 주 vs 지난 주
- 이번 분기 vs 지난 분기

비교 모드에서는 변화율(↑↓ %)이 함께 표시되어 추세 변화를 빠르게 파악할 수 있습니다.

## 알림 설정

특정 지표가 임계치를 넘으면 알림을 받을 수 있습니다.

1. 위젯 우상단 **알림 설정** 버튼
2. 임계치 입력 (예: "PII 탐지 일 100회 이상")
3. 알림 채널 선택 (이메일·웹훅·Slack 등)
4. **저장**

## 보고서 내보내기

월간·분기 보고서를 PDF/Excel로 내보낼 수 있습니다.

1. 우상단 **보고서 내보내기** 버튼
2. 형식 선택 (PDF / Excel)
3. 기간 선택
4. **다운로드**

!!! note "보고서 내보내기 옵션 캡처는 별도 갱신 예정"
    "보고서 내보내기" 클릭 시 노출되는 형식·기간 선택 모달 캡처는 다음 회차에 보강 예정입니다.

서비스 변경 이력은 별도 화면에서 관리됩니다.

![서비스 변경 이력 — 거버넌스 정책·서비스 구성 변경 추적](images/admin-gov-audit-tracking.png)

## 운영 권장사항

- **월 1회 정기 검토** — 운영팀·보안팀이 함께 대시보드를 검토하고 이상 항목을 처리
- **분기 가중치 재조정** — 외부 규제 변화·내부 사고 이력을 반영해 가중치 재설정
- **임계치는 환경에 맞게** — 사용자 수가 늘면 절대값(예: "PII 100회") 대신 비율(예: "전체 메시지의 1% 이상")로 임계치 설정 검토
- **보고서 자동화** — 분기 보고서는 스케줄러에 등록해 자동 발송
