# 솔루션 개요

{{product.name}}은(는) 사용자가 자연어로 데이터에 접근하고 업무를 자동화하는 기업용 AI 솔루션입니다.

## 주요 기능

- 자연어 기반 질의응답
- 지식베이스 관리
- 에이전트플로우 (Agentflow) 설계
- 관리자 대시보드를 통한 운영 모니터링

## 시스템 요구사항

| 항목 | 권장 사양 |
|---|---|
| 브라우저 | Chrome, Edge 최신 버전 |
| 화면 해상도 | 1920 x 1080 이상 |
| 네트워크 | 솔루션 도메인({{product.domain}})에 대한 HTTPS 접근 |
