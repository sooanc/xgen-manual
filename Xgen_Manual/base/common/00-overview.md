# 솔루션 개요

{{product.name}}은(는) 기업 환경에 최적화된 생성형 AI 플랫폼입니다. 맞춤형 AI 서비스의 설계부터 구축·운영·관리까지 통합 지원하며, 기업 업무에 필요한 Agent 기반 AI 서비스를 빠르고 안전하게 구현할 수 있습니다.

자연어 기반의 업무 자동화 환경을 제공하여, 사용자는 복잡한 개발 지식 없이도 다양한 AI 업무 흐름을 설계하고 실행할 수 있습니다.

## 주요 특징

### 1. Agent 기반 AI 서비스 구성

사용 목적에 맞는 Agent 를 생성하고 조합하여 업무 자동화 및 AI 기반 실행 환경을 구성할 수 있습니다.

- 업무별 Agent 생성
- Prompt 기반 실행 로직 설계
- Workflow 기반 업무 연결
- 다중 Agent 협업 구성 지원

### 2. 기업 환경에 최적화된 운영 구조

기업 내부 보안 정책과 운영 기준을 고려한 관리 기능을 제공합니다.

- 역할 기반 접근 제어 (RBAC)
- 배포 승인 및 운영 관리
- 감사 로그 및 실행 이력 관리
- 조직 기반 권한 체계 지원
- 온프레미스 및 클라우드 환경 대응

### 3. 자연어 기반 AI 업무 자동화

사용자는 자연어를 활용하여 업무를 실행하고 자동화할 수 있습니다.

- 문서 요약 및 생성
- 데이터 분석 및 질의
- 업무 절차 자동화
- 지식 검색 및 응답 생성
- 반복 업무 처리 지원

### 4. 다양한 AI 모델 및 시스템 연동

외부 시스템 및 AI 모델과의 유연한 연동을 지원합니다.

- 다양한 LLM 연동
- API 기반 시스템 연결
- 사내 데이터 연계
- RAG 기반 지식 검색 지원
- Tool 및 Function 연동 지원

## 제공 기능

- 자연어 기반 질의응답
- 지식베이스 관리
- 에이전트플로우 (Agentflow) 설계
- 관리자 대시보드를 통한 운영 모니터링
