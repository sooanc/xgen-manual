# Solution Overview

{{product.name}} is a generative AI platform optimized for the enterprise. It covers the full lifecycle of tailored AI services — design, build, operations, and management — so organizations can deliver Agent-based AI services for enterprise workflows quickly and securely.

With a natural-language work-automation environment, users can design and run a wide range of AI workflows without deep development knowledge.

## Key Highlights

### 1. Agent-based AI service composition

Create and combine purpose-built Agents to assemble a work-automation and AI execution environment.

- Per-task Agent creation
- Prompt-based execution logic design
- Workflow-based task orchestration
- Multi-Agent collaboration support

### 2. Enterprise-grade operational structure

Management capabilities aligned with internal security policies and operating standards.

- Role-Based Access Control (RBAC)
- Deployment approval and operations management
- Audit log and execution-history management
- Organization-based permission model
- On-premises and cloud deployment

### 3. Natural-language work automation

Users execute and automate tasks through natural language.

- Document summarization and generation
- Data analysis and Q&A
- Work-procedure automation
- Knowledge retrieval and response generation
- Repetitive-task handling

### 4. Diverse AI model and system integrations

Flexible integration with external systems and AI models.

- Multiple LLM integrations
- API-based system connections
- Internal data linkage
- RAG-based knowledge search
- Tool and Function integration

## Available Features

- Natural-language Q&A
- Knowledge base management
- Agentflow design (multi-step AI workflows)
- Operational monitoring via the admin dashboard
