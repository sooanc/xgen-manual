# 대시보드

로그인 직후 처음 노출되는 메인 화면입니다. 주소창 경로는 `{{product.domain}}/dashboard` 이며, 자주 쓰는 화면으로의 단축 진입과 본인 작업 현황 위젯이 모여 있습니다.

## 화면 구성

대시보드는 **상단 인사 카드**, **좌측 위젯 그리드**, **우측 고정 지원 패널** 세 부분으로 구성됩니다.

> 아래 위젯 그리드의 위험 등급 정책·금칙어 정책 현황 같은 거버넌스 위젯은 관리자 권한이 있는 계정에만 노출됩니다. 일반 사용자에게는 "자주 찾는 Agent TOP 3"·"인기 Agent 템플릿" 등 본인 작업 기반 위젯만 표시됩니다.

### 인사 카드와 빠른 이동

상단 카드는 "반갑습니다, OOO님! 오늘 업무는 어떤 Agent와 시작할까요?" 문구로 시작합니다. 그 아래 4개의 단축 버튼 중 일반 사용자에게 의미 있는 것은 **Agent 작업실**(`/main?view=agentflows`) 한 개입니다. 나머지 버튼(AI 모델관리, AI 거버넌스, 관리설정)은 관리자 권한이 있을 때만 동작합니다.

### 좌측 위젯 그리드

본인 권한과 사용 패턴에 따라 다음 위젯이 노출됩니다.

| 위젯 | 표시 내용 |
|---|---|
| 자주 찾는 Agent TOP 3 | 본인이 자주 호출한 에이전트 상위 3개 |
| 인기 Agent 템플릿 | 조회·복제가 많은 템플릿 |

### 우측 고정 패널

위젯 그리드와 별개로 항상 같은 자리에 노출됩니다.

| 패널 | 데이터 출처 | 더보기 링크 |
|---|---|---|
| 최신 업데이트 | 공지사항 최근 3건 | `/main?view=support-notices` |
| 자주 묻는 질문 | 조회수 순 상위 3건 | `/main?view=support-faq` |
| 최근 서비스 요청 목록 TOP 3 | 최근 서비스 요청 3건 | `/main?view=support-qna&intent=service-request` |
| 내 문의 | 본인이 작성한 1:1 문의 3건 | `/main?view=support-qna` |

각 패널 우상단의 **더보기 >** 링크로 해당 영역의 전체 목록으로 이동합니다.

## 위젯 커스터마이징

- **숨기기**: 위젯 카드 우상단 버튼으로 개별 위젯을 숨길 수 있습니다.
- **순서 변경**: 위젯 카드를 **드래그**해 원하는 자리에 놓습니다 (`@dnd-kit/sortable` 기반).
- **위젯 추가**: 상단 **위젯 추가** 드롭다운에서 숨겨놓은 위젯을 다시 표시합니다.
- **초기화**: 상단 **초기화** 버튼으로 출고 상태(기본 위젯 목록·순서)로 되돌립니다.

위 설정은 본인 계정에 한해 저장되므로 다른 사용자에게 영향을 주지 않습니다.

## 활용 순서

1. 로그인 후 `/dashboard`로 자동 진입합니다. 다른 화면에서 돌아오려면 좌상단 **XGEN** 로고를 클릭합니다.
2. **자주 찾는 Agent TOP 3** 위젯을 통해 한 번에 익숙한 에이전트로 들어갑니다.
3. 우측 패널의 **최신 업데이트**·**자주 묻는 질문**을 먼저 훑어본 뒤 작업을 시작하는 흐름을 권장합니다 — 시스템 변경이나 흔한 문제를 미리 인지할 수 있습니다.
4. 안 쓰는 위젯은 카드 우상단 **숨기기**로 정리하고, 필요해지면 상단 **위젯 추가**로 다시 표시합니다.
5. 위젯 순서가 마음에 들지 않으면 카드를 드래그해 원하는 자리에 놓고, 전부 되돌리려면 상단 **초기화**를 누릅니다.

## 자주 묻는 문제

- **위젯이 비어 있어요** — 본인 작업 기록이 누적되면 점차 채워집니다. "자주 찾는 Agent TOP 3"은 본인이 호출한 에이전트가 있어야 노출됩니다.
- **다른 사용자와 위젯 구성이 달라요** — 위젯 표시·순서·숨김 설정은 개인별로 저장되므로 정상입니다.
- **마지막 접속 시간이 비어 있어요** — 우상단 사용자 이름 좌측에 노출되는 **마지막 접속** 표시가 없거나 본인 기억과 다르면 즉시 관리자에게 문의해 주세요(보안 이상 가능성).
