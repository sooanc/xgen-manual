# 대시보드

로그인 직후 처음 노출되는 메인 화면입니다. 주소창 경로는 `{{product.domain}}/dashboard` 이며, 자주 쓰는 화면으로의 단축 진입과 본인 작업 현황 위젯이 모여 있습니다.

> `/dashboard` 한 경로지만, **로그인한 계정의 역할·권한에 따라 노출되는 위젯과 단축 버튼이 달라집니다.** 일반 사용자와 Agent 개발자가 보는 모습이 다르며, 시스템 관리자·거버넌스 담당자가 보는 모습은 [관리자 매뉴얼 · 대시보드](../admin/30-dashboard.md) 챕터에서 별도로 다룹니다.

## 화면 구성 (공통)

대시보드는 **상단 환영 메시지**, **좌측 위젯 그리드**, **우측 고정 지원 패널** 세 부분으로 구성됩니다. 이 골격은 모든 역할에서 동일하며, 안에 채워지는 위젯·버튼이 권한에 따라 달라집니다.

![대시보드 전체 화면 — 상단 환영 메시지, 좌측 위젯 그리드, 우측 고정 패널이 한 화면에 보이는 기본 레이아웃](images/dashboard-overview.png)

### 환영 메시지와 빠른 이동

상단 카드는 "반갑습니다, OOO님! 오늘 업무는 어떤 Agent와 시작할까요?" 문구로 시작합니다. 그 우측에 2개의 단축 버튼이 노출되며, 본인 권한이 있는 버튼만 동작합니다.

| 버튼 | 이동 위치 | 일반 사용자 | Agent 개발자 |
|---|---|---|---|
| Agent 작업실 | `/main?view=agentflows` | 동작 (채팅 진입) | 동작 (제작 진입) |
| 관리 설정 | `/admin` | 비활성 | 비활성 |

> **관리 설정** 버튼이 활성화된 모습은 [관리자 매뉴얼 · 대시보드](../admin/30-dashboard.md) 챕터에서 확인하세요.

### 우측 고정 패널

위젯 그리드와 별개로 항상 같은 자리에 노출됩니다.

| 패널 | 데이터 출처 | 더보기 링크 |
|---|---|---|
| 최신 업데이트 | 공지사항 최근 3건 | `/main?view=support-notices` |
| 자주 묻는 질문 | 조회수 순 상위 3건 | `/main?view=support-faq` |
| 최근 서비스 요청 목록 TOP 3 | 최근 서비스 요청 3건 | `/main?view=support-qna&intent=service-request` |
| 내 문의 | 본인이 작성한 1:1 문의 3건 | `/main?view=support-qna` |

각 패널 우상단의 **더보기 >** 링크로 해당 영역의 전체 목록으로 이동합니다.

## 일반 사용자 뷰

회사가 미리 배포해둔 Agent를 **채팅으로 활용**하는 사용자의 메인 화면입니다. 상단 부제는 **"Agent 활용 Dashboard"** — "나에게 공유된 Agent를 쉽게 실행하고, 서비스 요청 현황과 평가를 한눈에 확인할 수 있습니다." 안내가 함께 표시됩니다.

| 위젯 | 표시 내용 |
|---|---|
| 자주 찾는 Agent TOP 3 | 본인이 자주 호출한 에이전트 상위 3개 |
| 인기 Agent 템플릿 | 조회·복제가 많은 템플릿 |
| 공유 받은 Agent | 다른 사용자가 본인에게 공유한 에이전트 목록 |
| 내가 남긴 평가 | 본인이 작성한 종합 만족도 / 피드백 참여율 요약 |

위젯이 비어 있다면 본인 작업 기록이 누적되면 점차 채워집니다.

![최수안(일반사용자) 계정으로 본 /dashboard — "Agent 활용 Dashboard" 부제 아래에 자주 찾는 Agent TOP 3·인기 Agent 템플릿·공유 받은 Agent·내가 남긴 평가 위젯이 노출된 모습](images/dashboard-standard.png)

## Agent 개발자 뷰

자기 업무를 자동화할 Agent를 **직접 설계·배포**하는 사용자의 메인 화면입니다. 상단 부제는 일반 사용자와 동일한 **"Agent 활용 Dashboard"** 이며, 위젯 그리드에 **본인이 제작한 에이전트플로우 현황**과 **공유 자산 현황** 위젯이 추가됩니다.

| 위젯 | 표시 내용 |
|---|---|
| 자주 찾는 Agent TOP 3 | (공통) 본인이 자주 호출한 에이전트 |
| 인기 Agent 템플릿 | (공통) 조회·복제가 많은 템플릿 |
| 공유 받은 Agent | (공통) 다른 사용자가 본인에게 공유한 에이전트 |
| 내가 남긴 평가 | (공통) 본인 평가 요약 |
| 내 에이전트플로우 현황 | 본인 소유 에이전트플로우의 전체·공유됨·배포됨 건수와 최근 항목 |
| 공유 자산 현황 | 본인이 공유한 도구·지식 컬렉션 등 자산 요약 |

내 에이전트플로우 현황 위젯이 비어 있으면 첫 에이전트 배포 직후 자료가 누적될 때까지 비어 있을 수 있습니다. [에이전트 운영](13-agentflow-operations.md) 진입 경로의 단축으로 활용하세요.

![최수안(에이전트 개발자) 계정으로 본 /dashboard — 일반 사용자 위젯에 더해 '내 에이전트플로우 현황' '공유 자산 현황' 위젯이 노출된 모습](images/dashboard-developer.png)

## 위젯 커스터마이징

- **숨기기**: 위젯 카드 우상단 버튼으로 개별 위젯을 숨길 수 있습니다.
- **순서 변경**: 위젯 카드를 **드래그**해 원하는 자리에 놓습니다 (`@dnd-kit/sortable` 기반).
- **위젯 추가**: 상단 **위젯 추가** 드롭다운에서 숨겨놓은 위젯을 다시 표시합니다.
- **초기화**: 상단 **초기화** 버튼으로 출고 상태(기본 위젯 목록·순서)로 되돌립니다.

위 설정은 본인 계정에 한해 저장되므로 다른 사용자에게 영향을 주지 않습니다.

전체 페이지를 스크롤한 모습은 다음과 같습니다.

![대시보드 전체 스크롤 — 좌측 위젯 그리드와 우측 패널이 끝까지 펼쳐진 모습](images/dashboard-full.png)

## 활용 순서

1. 로그인 후 `/dashboard`로 자동 진입합니다. 다른 화면에서 돌아오려면 좌상단 **XGEN** 로고를 클릭합니다.
2. **자주 찾는 Agent TOP 3** 위젯을 통해 한 번에 익숙한 에이전트로 들어갑니다.
3. 우측 패널의 **최신 업데이트**·**자주 묻는 질문**을 먼저 훑어본 뒤 작업을 시작하는 흐름을 권장합니다 — 시스템 변경이나 흔한 문제를 미리 인지할 수 있습니다.
4. 안 쓰는 위젯은 카드 우상단 **숨기기**로 정리하고, 필요해지면 상단 **위젯 추가**로 다시 표시합니다.
5. 위젯 순서가 마음에 들지 않으면 카드를 드래그해 원하는 자리에 놓고, 전부 되돌리려면 상단 **초기화**를 누릅니다.

## 자주 묻는 문제

- **위젯이 비어 있어요** — 본인 작업 기록이 누적되면 점차 채워집니다. "자주 찾는 Agent TOP 3"은 본인이 호출한 에이전트가 있어야 노출됩니다.
- **다른 사용자와 위젯 구성이 달라요** — 위젯 표시·순서·숨김 설정은 개인별로 저장되므로 정상입니다. 또한 권한(일반 사용자/Agent 개발자)에 따라 노출되는 위젯 목록 자체가 다릅니다. 일반 사용자와 Agent 개발자는 같은 부제("Agent 활용 Dashboard") 를 공유하지만 Agent 개발자에게는 본인 제작 자산 위젯이 추가됩니다.
- **마지막 접속 시간이 비어 있어요** — 우상단 사용자 이름 좌측에 노출되는 **마지막 접속** 표시가 없거나 본인 기억과 다르면 즉시 관리자에게 문의해 주세요(보안 이상 가능성).
