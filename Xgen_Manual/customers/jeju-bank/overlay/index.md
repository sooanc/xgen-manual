# 솔루션 가이드

본 매뉴얼은 {{product.name}} 솔루션의 사용자 및 관리자 운영 가이드입니다. 두 가지 방식으로 활용하실 수 있습니다.

## 빠른 시작 — [업무 가이드](tasks/index.md)

자기 역할과 **하고 싶은 일** 기준으로 필요한 챕터를 바로 찾아갈 수 있습니다. 메뉴를 외울 필요 없이 다음 4개 역할 중 본인에 맞는 곳으로 진입하세요.

- 일반 사용자 — Agent 채팅·기술지원
- Agent 개발자 — Agent 설계·배포·승인 받기
- 시스템 관리자 — 사용자·권한·LLM·시스템 운영
- 거버넌스 담당자 — 승인·통제 정책·감사 <!-- require_view: gov-monitoring -->

→ [업무 가이드 페이지 열기](tasks/index.md)

## 메뉴별 레퍼런스

솔루션 메뉴 구조 그대로 상세 정보를 찾고 싶을 때 사용합니다.

- [**공통**](common/) — 솔루션 개요, 용어 정의 등 사용자/관리자 모두에게 필요한 내용
- [**사용자**](user/) — 일반 사용자·개발자가 사용하는 화면 (Agent 작업실 영역)
- [**관리자**](admin/) — 시스템 관리자·거버넌스 담당자가 사용하는 화면 (관리 설정 영역) <!-- require_view: gov-monitoring -->
- [**관리자**](admin/) — 시스템 관리자가 사용하는 화면 (관리 설정 영역) <!-- require_view: no-governance -->
