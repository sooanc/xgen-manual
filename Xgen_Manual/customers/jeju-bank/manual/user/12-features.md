# 사용자 기능 가이드

본 챕터는 **Agent 작업실** 의 주요 기능을 영역별로 정리합니다. 각 항목은 화면 진입 경로와 함께 사용 절차를 안내합니다.

!!! info "스크린샷 보강 예정"
    본 챕터의 일부 절은 다음 회차 캡처에서 스크린샷을 추가합니다. 현재는 텍스트 기반 화면 설명·라벨만 정리되어 있습니다.

## Agent 제작 · 채팅으로 워크플로우 만들기

자연어 대화로 에이전트플로우 노드를 자동 구성하는 기능입니다.

**진입 경로**

좌측 사이드바 **Agent 제작 → Agent 설계** → 캔버스 상단 **채팅으로 만들기**

**사용 절차**

1. 캔버스 진입 후 상단 **채팅으로 만들기** 클릭
2. 만들고 싶은 흐름을 자연어로 설명 — 예: *"사내 규정 문서를 검색해 답변하는 챗봇"*
3. AI 가 노드 구성을 제안하면 검토 후 **수락** → 캔버스에 즉시 반영
4. 생성된 노드는 일반 캔버스 편집과 동일하게 수정 가능

**진행 표시**

응답 스트리밍 중 입력 커서가 표시되며, 단계별 진행 상태가 좌측 패널에 노출됩니다. 진행 중 응답은 *중단* 버튼으로 종료할 수 있습니다.

## Agent 제작 · 사이드바 메뉴 구성

좌측 사이드바 **Agent 제작** 섹션은 다음 순서로 구성됩니다.

| 순서 | 메뉴 | 용도 |
|---|---|---|
| 1 | Agent 기획 | 신규 에이전트 기획서 등록·검토 |
| 2 | Agent 제작 소개 | 캔버스 진입 안내 |
| 3 | Agent 설계 | 캔버스에서 에이전트 직접 제작 |
| 4 | Agent 목록 | 본인이 만든 에이전트플로우 관리 |
| 5 | Agent 운영 설정 | 배포·스케줄 등 운영 옵션 |
| 6 | Agent 품질 평가 | 응답 품질 자동 평가 |
| 7 | Agent 프롬프트 | 프롬프트 템플릿 관리 |

**Agent 기획** 메뉴는 **Agent 제작 섹션 최상단** 에 위치합니다. 새 에이전트 작업을 시작할 때 기획서를 먼저 등록한 뒤 캔버스로 이동하는 흐름을 권장합니다.

## 지식 관리 · 컬렉션 사전 (Dictionary)

지식 컬렉션에 동의어·금칙어·도메인 용어 묶음을 *사전* 단위로 등록해 검색 정확도를 높일 수 있습니다.

**진입 경로**

좌측 사이드바 **지식관리 → 지식 컬렉션 → (컬렉션 선택) → 사전 탭**

**사용 절차**

| 동작 | 절차 |
|---|---|
| 사전 목록 조회 | 컬렉션 상세 → **사전** 탭 |
| 사전 생성 | 사전 탭 우상단 **+ 사전 추가** → 이름·설명 입력 |
| 용어 일괄 등록 | 사전 상세 → **벌크 가져오기** → CSV 업로드 |
| 컬렉션에 적용 | 컬렉션 생성·편집 모달 → *사전 선택* 드롭다운 |

CSV 업로드는 *용어, 동의어, 비고* 컬럼 구조로 작성합니다. 헤더 행 포함 여부는 업로드 모달에서 선택할 수 있습니다.

## 지식 관리 · 문서 업로드 PII 사전 스캔

문서 업로드 시 개인정보(PII) 를 자동 감지·마스킹하는 사전 스캔이 실시간 진행 표시로 동작합니다.

**처리 흐름**

1. **지식관리 → 지식 컬렉션 → (컬렉션 선택) → 업로드** 진입
2. 파일 선택 → 업로드 옵션에서 *PII 스캔* 활성화 확인 (기본 ON)
3. **업로드 시작** → 상단에 *PII 사전 스캔 진행 단계* 가 실시간 표시
4. 스캔 완료 후 검출된 PII 가 마스킹 처리된 상태로 컬렉션에 저장

**처리 시간 안내**

- 30 초마다 진행 상태 메시지가 갱신됩니다 (네트워크 중단 방지용 heartbeat).
- 단일 파일 최대 처리 시간: 10 분. 대용량 문서는 분할 업로드를 권장합니다.

## 지식 관리 · 온톨로지 빌드

컬렉션 단위로 *개념·관계* 메타데이터를 자동 추출해 검색 품질을 향상시킵니다.

**진입 경로**

컬렉션 상세 → **온톨로지** 탭 → **빌드 시작**

**빌드 도중 취소**

빌드 진행 중 **취소** 버튼을 누르면 즉시 종료됩니다. 취소 후 잠시 *처리 중* 상태로 표시될 수 있으나 자동으로 *종료* 로 정리됩니다.

## 채팅 · 사이드바 빠른 이동

사이드바 헤더와 대시보드 환영 카드 우측에 **Teams** / **Canvas** 빠른 이동 버튼이 노출됩니다.

| 위치 | 형태 |
|---|---|
| 사이드바 헤더 | 작은 outline pill 버튼 |
| 대시보드 환영 카드 우측 | 큰 그라데이션 버튼 |

- **Teams** — 팀 협업 채팅 공간으로 이동 (`/teams`)
- **Canvas** — 에이전트 설계 캔버스로 직접 이동 (`/canvas`). 권한이 있는 사용자에게만 노출.

## 대시보드 · 지원 페이지 빠른 진입

대시보드 우측 패널의 *최신 업데이트 / 자주 묻는 질문 / 관리자 문의* 항목을 클릭하면 해당 게시물이 자동으로 펼쳐진 상태로 페이지에 진입합니다.

| 위젯 항목 | 진입 결과 |
|---|---|
| 자주 묻는 질문 (FAQ) | FAQ 페이지가 해당 질문이 펼쳐진 상태로 열림 |
| 공지사항 | 공지 페이지가 해당 공지로 스크롤·강조된 상태로 열림 |
| 관리자 문의 | 1:1 문의 페이지가 해당 문의 상세 모달이 열린 상태로 진입 |

## 알림 센터

사이드바 헤더의 알림 아이콘 옆 카운트가 다음 시점에 즉시 갱신됩니다.

- 페이지 이동
- 사이드바 메뉴 전환
- 브라우저 창 포커스 복귀
- 모달 열기·닫기

별도 새로고침 없이 본인의 최근 알림 상태를 확인할 수 있습니다.
