# 로그인 ({{customer.name}} SSO 환경)

> **{{customer.name}} 안내:** 본 환경은 사내 SSO(통합인증)를 통한 자동 로그인이 적용됩니다.

## 로그인 절차

1. 사내 인트라넷에 로그인된 상태에서 {{product.domain}}에 접속합니다.
2. 별도 입력 없이 자동으로 로그인됩니다.
3. 자동 로그인이 동작하지 않는 경우 **XGen 관리자**({{vars.support_email}})로 문의하세요.

## 로그아웃

자동 로그인 환경에서는 명시적 로그아웃이 인트라넷 세션 종료와 연동됩니다. 화면 우측 상단의 **로그아웃** 버튼은 솔루션 세션만 종료하며, 사내 인트라넷 세션은 유지됩니다.

## 보안 안내

- 동일 PC에서 다른 사용자 계정으로 접속하지 마십시오.
- 자리를 비울 때는 화면 잠금(Win + L)을 사용하시고, 솔루션 로그아웃은 별도로 수행하지 않아도 인트라넷 세션 종료 시 자동 처리됩니다.
