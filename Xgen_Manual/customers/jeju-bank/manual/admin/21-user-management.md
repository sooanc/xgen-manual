# 사용자 관리

## 사용자 추가

1. 좌측 메뉴에서 **관리 > 사용자 관리**를 선택합니다.
2. 우측 상단의 **+ 사용자 추가** 버튼을 클릭합니다.
3. 다음 정보를 입력합니다.
    - 이름
    - 이메일
    - 부서
    - 역할(Role)
4. **저장**을 클릭합니다.

## 권한 부여

사용자 목록에서 대상 행의 **역할** 드롭다운을 변경하면 즉시 반영됩니다.

| 역할 | 권한 |
|---|---|
| 일반 사용자 | 솔루션 사용 |
| 운영자 | 지식베이스 관리 |
| 관리자 | 전체 관리 권한 |

## 사용자 비활성화

삭제 대신 비활성화를 권장합니다. 비활성화된 사용자는 로그인할 수 없으나 기존 작업 이력은 보존됩니다.
